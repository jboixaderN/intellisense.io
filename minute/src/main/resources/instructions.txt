The git project has already the jar file but i can case you'd like to create you can do it using maven:
mvn clean compile install
This command will create a jar file (minute-0.0.1-SNAPSHOT.jar) into target folder.

For running this jar:
1. Go to the jar folder location
2  Type the following command: java -jar minute-0.0.1-SNAPSHOT.jar
